# Ownership & RBAC

## Rôles types
- Admin Dynatrace (plateforme)
- Observability Engineer / SRE
- Read-only (auditeurs, métiers)
- On-call (N2/N3)

## Matrice (exemple)
- Création tokens : Admin
- Dashboards/alerting : SRE
- Définition SLO : SRE + Service Owner
- Changements agent/operator : Plateforme + Change Mgmt
