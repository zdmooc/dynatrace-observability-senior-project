# Dashboards & alerting

## Dashboards “socle”
- Santé globale (golden signals)
- Disponibilité / SLO
- Performance (latence p95/p99)
- Erreurs (5xx, exceptions)
- Capacité (CPU, mémoire, saturation)

## Alerting
- Baselines / anomaly detection
- Réduction du bruit (déduplication, maintenance windows)
- Routing par MZ / tags
