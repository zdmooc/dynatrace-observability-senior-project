# Documentation

## Parcours recommandé
1. **01-cadrage/** : périmètre, exigences, responsabilités
2. **02-installation/** : installation détaillée Dynatrace (Managed/ActiveGate/OneAgent/K8s)
3. **03-standards/** : tags, management zones, naming, ownership
4. **04-instrumentation/** : middlewares, logs, traces, OTEL (si applicable)
5. **05-dashboards-alerting/** : dashboards, alertes, baselines, réduction bruit
6. **06-sli-slo/** : SLI/SLO, error budget, burn rate
7. **07-reporting/** : rapport mensuel, KPI, tendances, capacité
8. **08-operations/** : runbooks N3, incident, crise, RCA
